# Custom Invoice Generator App

Custom Invoice Generator App is a Flutter-based mobile application designed to create customizable invoices on the go. With this app, users can easily add items to their invoices, generate PDFs, and share them with clients seamlessly. The app provides a user-friendly interface and efficient features to streamline the invoicing process.

## Features

- **Customizable Invoices**: Users can add custom items to their invoices, including descriptions, quantities, rates, and taxes.
- **PDF Generation**: The app generates professional-looking PDF invoices from the entered data, ensuring easy sharing and printing.
- **Item Management**: Users can add, edit, and remove items from their invoices for flexibility and accuracy.
- **Share Functionality**: Invoices can be shared directly from the app via various communication channels like email, messaging apps, or cloud storage services.

## Technologies Used

- **Flutter**: The app is built using the Flutter framework, enabling cross-platform compatibility and a native-like user experience.
- **Dart**: Dart programming language is used for coding the app logic and UI components.
- **Android Studio**: The development environment for building and testing the app is Android Studio, providing a robust set of tools for Flutter development.
